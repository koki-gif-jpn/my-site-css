大切にしている考え方

私たちのプログラムは、
「体験から学ぶこと」と「人と人との関わりの中で生まれる変化」を軸にしています。
このページでは、それらの土台となっている主要な理論をご紹介します。

このページについて

研修や合宿などの場は、多くの場合、日常とは少し違う「非日常」の環境です。
そこでのふるまいや何気ない反応には、そのチームや個人がふだん過ごしている 関係・文脈・相互作用のクセが表れます。

私たちは、こうした場で起きる小さなサインから学びを取り出し、
対話を通して “気づきが行動へつながる” プロセスを大切にしています。

以下に紹介する理論は、正解を押しつけるためのものではなく、現場を一緒に眺めるためのレンズです。
必要に応じて使い分けながら、場に合った形でサポートしていきます。

体験から学ぶこと（コルブの経験学習モデル / Kolb’s Experiential Learning Model）

体験学習サイクルとしても知られる学習モデルで、
経験 → 観察 → 概念化 → 実験 の4つのプロセスを循環させながら、学びを深めていく理論です。

活動中に感じた違和感や気づきは、ただの“出来事”ではなく、次の行動に向けたヒントになります。
私たちは、その場で起きた体験を一緒に振り返りながら、意味づけし、日常で試せる形へ橋渡しします。

場の動きを読み解く（グループダイナミクス / Group Dynamics）

人のふるまいは、性格だけで決まるわけではありません。
立つ位置・視線・沈黙・反応など、場の雰囲気や相互作用から自然と生まれる動きが大きく関係します。

こうした小さなサインを丁寧に観察し、
「いまこのチームで何が起きているのか？」を一緒に見つめることで、関係の理解が深まり、より動きやすい状態が生まれます。

関係の質から始まる変化（成功循環モデル / Success Cycle Model）

関係の質 → 思考の質 → 行動の質 → 結果の質という流れを示すモデルです。
「結果を変えたい」という思いは、まず関係性の整いから始まります。

安心して話せる関係ができると、思考が柔らかくなり、行動が前向きになり、結果が自然と変化していきます。
私たちのプログラムは、この循環を意識した関わりを大切にしています。

チームの育ち方（タックマンモデル / Tuckman’s Stages of Group Development）

チームが育つプロセスを
形成期 → 混乱期 → 統一期 → 遂行期 の4段階で説明するモデルです。

混乱やズレは“問題”ではなく、チームが成長していくうえで自然に起こるもの。
その揺れをどう扱うかによって、関係の質も成果の質も大きく変わります。

行動の改善サイクル（PDCAサイクル / Plan–Do–Check–Act）

行動を計画し、やってみて、振り返り、改善するためのシンプルなフレームです。

対話で生まれた気づきを、次の行動へつなぐための“橋渡しのツール”として活用しています。

理論は正解ではなく、現場を見るためのレンズ

理論は “当てはめるため” ではなく、
その場で起きていることを一緒に眺め、理解を深めるための 補助線 です。

どの理論を使うかではなく、
「このチームに今必要な視点は何か？」を丁寧に選びながら、人と場に向き合うことを大切にしています。
