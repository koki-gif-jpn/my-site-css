大事にしている理論

組織開発事務所では、プログラムづくりの土台として「体験から学ぶ」ことを大切にしています。
ここでは、その背景にある理論・フレームワークの概要をご紹介します。

このページについて

研修や合宿などの場は、多くの場合「非日常」の時間です。
そこでの体験の中には、「もっと動けたんだろう」「もっと言葉が出たんだろう」と後から感じるような、 その人本来の力や可能性がにじむ瞬間がたくさんあります。

けれど、日常の現場に戻ると、いつもの習慣や空気がすぐに戻ってきてしまうこともあります。
会議室の話し合いだけでは決して見えませんが、日常の空間で表れている “普段は意識されていない行動のクセ” が、チームの協働に大きく影響していることも少なくありません。

だからこそ私たちは、「体験 → 対話 → 実践 → 体験…」という循環を意識して場を設計しています。
とくに、次の３つのフレーム 「コンフォートゾーン／学習ゾーンの枠組み」 「組織の成功循環モデル」 「経験学習サイクル（David Kolb）」を大事にしながら、 非日常の体験と日常の現場をつなぐことを目指しています。

紹介する主な理論・フレームワーク

体験学習とは

体験学習は、「実際にやってみた経験」から学びを引き出す考え方です。
活動そのものだけでなく、その後の振り返りや対話、日常での実践までを含めてひとつの学びのプロセスとして扱います。

会議室での話し合いだけでは見えにくい、日常の空間でのちょっとした行動や関わり方が、 体験の場では自然に表れてきます。
「あのとき気づけていたら、もっと動けたんだろう」「もっと言葉が出たんだろう」と感じる瞬間を、 次の一歩につなげていくことが体験学習のねらいです。

（ここに「体験 → 振り返り → 対話 → 実践」などの全体イメージ図を配置する予定）

コンフォートゾーンと学習ゾーンの考え方

もともとは「コンフォート・ゾーン／ストレッチ・ゾーン／パニック・ゾーン」の３つのゾーンとして語られてきた考え方から、 組織開発の文脈では「コンフォート／フィアー／ラーニング／グロース」の４つのゾーンとして整理されることもあります。

安心していられるコンフォートゾーンと、不安が強すぎて動けなくなるパニック（フィアー）ゾーンのあいだにある、 “ちょうどよい挑戦”の領域がラーニングゾーンです。
私たちは、参加される方がこのラーニングゾーンで試行錯誤できるよう、活動の強度や声かけを調整しながら場をつくっています。

（３つ／４つのゾーンの概念図を入れるスペース）

組織の成功循環モデル

組織の成功循環モデルは、「結果の質」だけを追いかけるのではなく、
その土台となる「関係の質」「思考の質」「行動の質」に目を向けることで、 結果の質につなげていく考え方です。

プログラムでは、とくに「関係の質」を丁寧に扱います。
日常の場面でどのような関わり方・対話が生まれているかを体験の中で見つめ直し、 そこで得た気づきが思考の質・行動の質の変化につながるように設計しています。

（関係の質 → 思考の質 → 行動の質 → 結果の質、の循環図スペース）

経験学習サイクル（David Kolb）

David Kolb による経験学習サイクルは、
「経験 → 振り返り → 気づき（概念化） → 実践」という流れを通して学びを深めていくモデルです。

私たちは、このサイクルを「体験 → 対話 → 実践 → 体験…」という形で現場に落とし込んでいます。
非日常のプログラムでの経験を振り返り、そこで得た気づきを日常の実践につなげる。
そして、その実践が次の「経験」をつくり、再び振り返りと対話につながっていく――そんな循環を支えるための枠組みとして活用しています。

（円形のサイクル図：経験 → 振り返り → 気づき → 実践 → 次の経験）
